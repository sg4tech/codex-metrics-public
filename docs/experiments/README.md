# Experiments

Use this folder for small, explicit product or workflow experiments that need a before/after record.

Recommended shape for each experiment:

- one markdown file per experiment
- link to the related hypothesis in the matching `docs/product-hypotheses/H-xxx.md` file and keep the index link handy if needed
- include a baseline snapshot before the change
- describe what changes, what to measure, and when to re-evaluate

Keep experiments operational and small. If a result becomes stable enough to act on by default, promote the conclusion into `docs/product-framing.md` and keep the experiment file as historical evidence.
